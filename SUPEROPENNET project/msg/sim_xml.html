유사 XML

다음과 같은 텍스트를 메시지 본문에 작성하면 다음과 같이 변합니다.

[엔터] : 다음 행으로 글을 넘깁니다. 단순히 엔터(키보드에서) 키를 넣은 경우는 삭제됩니다.

*예시 : 안녕?[엔터] 난 사람이야. -> 안녕?(다음 줄)난 사람이야.

[큰글씨], [/큰글씨] : [큰글씨] 태그로 열고, [/큰글씨] 태그로 닫습니다. 글씨를 크게 만듭니다. 자동으로 엔터 키 처리됩니다.

*예시 : [큰글씨]안녕?[/큰글씨]난 사람이야 ->

안녕?

난 사람이야.

[빨강], [초록], [파랑], [검정] : 각 태그로 색을 변경합니다. 기본 색은 검정이며, 닫는 태그는 없습니다.

*예시 : 안녕? [빨강]난 [파랑]사람[초록]이[검정]야. -> 안녕? 난 사람이야.
